Capstone Project README file

Housing Prices: Advanced Regression Techniques

The Ames Housing dataset was compiled by Dean De Cock for use in data science education.

Additional information about the dataset can be found in the data_description.txt 

All code written for this project is contained within a Jupyter notebook. In order to open the file you must run 'jupyter notebook NAME_OF_NOTEBOOK'

All code  written for this project was tested and run against python version 2.7.12.final.0

Additional libraries used for this project include:

matplotlib 1.5.3
numpy 1.11.1
pandas 0.18.1
scikit-learn 0.17.1
scipy 0.18.1
seaborn 0.7.1
visuals.py
visuals.pyc
